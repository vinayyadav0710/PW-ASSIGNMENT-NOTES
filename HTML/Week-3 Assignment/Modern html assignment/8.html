WHAT IS HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScripT

WHAT IS SEMANTIC HTML

Semantic HTML is the use of HTML markup to reinforce the semantics, or meaning, of the information in web pages and web applications rather than merely to define its presentation or look. Semantic HTML is processed by traditional web browsers as well as by many other user agents.

FIVE COMMONLY USE SEMANTIC TAGS

  • <header>: This tag represents the introductory content or a group of introductory content in a web page. It typically contains the website's logo, navigation menus, headings, and other elements related to the top section of the page.
  • <nav>: The <nav> tag denotes a section of the page that contains navigation links. It is used to define a block of navigation elements, such as menus or lists of links, allowing users to move between different pages or sections within the website.
  • <main>: The <main> tag represents the main content of a document or webpage. It should typically contain unique content that is directly related to or expands upon the central topic of the page. Each page should ideally have only one
    tag.
  • <article>: The <article> tag defines a self-contained piece of content that is meant to be independently distributable or reusable. It could be a blog post, news article, forum post, or any other stand-alone content item.
  • <footer>: The <footer> tag is used to mark the footer or the bottom section of a webpage. It typically contains information about the author, copyright notices, contact details, and links to related documents or resources.
